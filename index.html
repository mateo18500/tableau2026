❓ Guide d'utilisation — Tableau de Garde
🗂️ Saisie du personnel
Cliquer sur un champ nom → sélectionner dans la liste déroulante ou taper un nom libre · Saisir le numéro DECT ou téléphone dans le champ à droite · Tout est sauvegardé automatiquement.
🔘 Les boutons de la barre
💾 Enregistrer Sauvegarde toutes les données saisies localement sur ce PC (persistant après fermeture).
↺ Réinitialiser Efface toutes les données et remet le tableau à zéro. Une confirmation est demandée.
📋 Listes de référence Modifier les noms disponibles dans les menus déroulants (ajouter / supprimer du personnel).
📄 Résumé garde Affiche un récapitulatif de toute l'équipe en poste. Boutons Copier et Imprimer disponibles.
🗺️ Carte secteurs Ouvre la carte géographique interactive des secteurs ambulances avec l'équipe en temps réel.
📟 Appel rapide Panel de contact instantané avec tous les DECT (régulateurs, ARM) et téléphones (ambulances).
🔄 Modèles & Rotation Sauvegarder des équipes types et les assigner à un jour de la semaine pour remplissage automatique.
🌙 Mode sombre Bascule l'interface en thème sombre. Préférence sauvegardée automatiquement.
📋 Listes de référence — comment modifier
1. Cliquer sur 📋 Listes de référence · 2. Trouver la catégorie (Régulateurs, ARM, IDE SMUR…) · 3. Ajouter ou supprimer des noms · 4. Cliquer Enregistrer. Les noms apparaissent ensuite dans les menus déroulants.
🔄 Modèles & Rotation — comment utiliser
1. Remplir tout le tableau · 2. Ouvrir Modèles & Rotation · 3. Saisir un nom (ex : Equipe A) et cliquer Sauvegarder le modèle · 4. Dans la rotation, assigner chaque modèle à un jour de la semaine · 5. Activer la rotation automatique → le tableau se pré-remplit selon le jour.
🗺️ Carte des secteurs — modifier les zones
Les contours des secteurs sont définis dans le tableau SECTEURS du fichier HTML (coordonnées GPS). Pour les modifier, utiliser geojson.io pour dessiner les zones et récupérer les coordonnées [lat, lng].
💊 Pharmacie de Garde
La liste des pharmacies affichée vient des données intégrées dans le fichier. Le bouton 🌐 Site officiel ouvre le site national pour trouver la pharmacie de garde en temps réel. Le sélecteur (Bourges / Vierzon / St-Amand) filtre la liste par zone.
💡 Raccourcis clavier
Échap Ferme n'importe quelle fenêtre ouverte (Résumé, Carte, Appel rapide, Rotation…).
Ctrl + P Imprimer la page complète (ou utiliser le bouton Imprimer dans le Résumé garde).

❓ Guide d'utilisation — Tableau de Garde
🗂️ Saisie du personnel
Cliquer sur un champ nom → sélectionner dans la liste déroulante ou taper un nom libre · Saisir le numéro DECT ou téléphone dans le champ à droite · Tout est sauvegardé automatiquement.
🔘 Les boutons de la barre
💾 Enregistrer Sauvegarde toutes les données saisies localement sur ce PC (persistant après fermeture).
↺ Réinitialiser Efface toutes les données et remet le tableau à zéro. Une confirmation est demandée.
📋 Listes de référence Modifier les noms disponibles dans les menus déroulants (ajouter / supprimer du personnel).
📄 Résumé garde Affiche un récapitulatif de toute l'équipe en poste. Boutons Copier et Imprimer disponibles.
🗺️ Carte secteurs Ouvre la carte géographique interactive des secteurs ambulances avec l'équipe en temps réel.
📟 Appel rapide Panel de contact instantané avec tous les DECT (régulateurs, ARM) et téléphones (ambulances).
🔄 Modèles & Rotation Sauvegarder des équipes types et les assigner à un jour de la semaine pour remplissage automatique.
🌙 Mode sombre Bascule l'interface en thème sombre. Préférence sauvegardée automatiquement.
📋 Listes de référence — comment modifier
1. Cliquer sur 📋 Listes de référence · 2. Trouver la catégorie (Régulateurs, ARM, IDE SMUR…) · 3. Ajouter ou supprimer des noms · 4. Cliquer Enregistrer. Les noms apparaissent ensuite dans les menus déroulants.
🔄 Modèles & Rotation — comment utiliser
1. Remplir tout le tableau · 2. Ouvrir Modèles & Rotation · 3. Saisir un nom (ex : Equipe A) et cliquer Sauvegarder le modèle · 4. Dans la rotation, assigner chaque modèle à un jour de la semaine · 5. Activer la rotation automatique → le tableau se pré-remplit selon le jour.
🗺️ Carte des secteurs — modifier les zones
Les contours des secteurs sont définis dans le tableau SECTEURS du fichier HTML (coordonnées GPS). Pour les modifier, utiliser geojson.io pour dessiner les zones et récupérer les coordonnées [lat, lng].
💊 Pharmacie de Garde
La liste des pharmacies affichée vient des données intégrées dans le fichier. Le bouton 🌐 Site officiel ouvre le site national pour trouver la pharmacie de garde en temps réel. Le sélecteur (Bourges / Vierzon / St-Amand) filtre la liste par zone.
